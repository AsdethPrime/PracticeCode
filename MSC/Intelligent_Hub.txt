Force device check in:
-> Open intelligent hub
-> Click on "This Device"
-> At the very end you should seen an option "Send data" which should ideally give you a message like "Request sent, syncing will take few minutes" 

Re-enroll device:
-> Open intelligent hub
-> Click on "This Device"
-> Click on "Enrollment"
-> This will have a "Reenroll device" option
This also has the following:
1) Mode: Managed device 
2) Server: 
3) Group ID:
4) Username: 

Testing internet connection from Intelligent hub:
-> Open intelligent hub 
-> Click on "Support"
-> Click on "Run Tests"
-> Check what test have been configured. Each test name can normally be clicked to cause this tests to run
